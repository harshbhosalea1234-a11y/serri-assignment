Serri Intern Assignment

Project Overview
I have integrated a Contacts system that allows an AI to search through a list of names and companies to answer user questions.

Answers to Assignment Questions
1. Scaling to 1 Million Contacts: I would use a "Vector Database" or "Elasticsearch" to keep searches fast.
2. Finding the Right Person: I would use "Semantic Search" so the AI understands meanings, not just exact words.
3. Current Limitations: This is a basic version that uses keyword matching; a real version would sync with a live database.

Data Analysis Insights (1M Rows)
- Peak Activity: Users are most active between 10 AM and 2 PM.
- Top Industries: E-commerce and Education use the most automation.
- Campaign Idea: Send "Abandoned Cart" reminders via WhatsApp to users who stop at the payment stage.
